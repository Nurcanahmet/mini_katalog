# 🛍️ Mini Katalog Uygulaması

Flutter ile geliştirilmiş basit bir e-ticaret katalog uygulaması.

## 📱 Özellikler

- Ürün listesi (GridView)
- Ürün detay sayfası
- Sepete ekle / çıkar
- Ürün arama ve filtreleme
- API'den gerçek zamanlı veri çekme

## 🛠️ Kullanılan Teknolojiler

- Flutter 3.41.4
- Dart
- http paketi
- Fake Store API

## 🚀 Çalıştırma Adımları

1. Repoyu klonla:
git clone https://github.com/Nurcanahmet/mini_katalog.git

2. Bağımlılıkları yükle:
flutter pub get

3. Uygulamayı çalıştır:
flutter run
